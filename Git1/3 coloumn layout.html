Welcome To My Three Column Responsive Layout

About this Layout

This Three coloumn Responsive Layout is very basic and most used layout

I have designed and developed this based on the knowledge of Front-End Development that I have gained from my project based experience and work based experience.

Why Three Coloumn Responsive layout

I have prioritized this layout mainly because this layout brings a smart and beautiful look with concise view and used by almost many websites to describe clearly about the description of contents

I have used CSS3 media queries for making the page viewed in 960px or less and 680px or less media screens

Media Queries check many things like width and height of viewport, device, resolution nad orientation(landscape or portrait)

How I have done this Layout

Firstly, I have designed whole page in a piece of paper and I choosed sublime which is my most favourite coding editor that helped me alot for making my hands-on practice possible. Later on, I have implemented the design into development using HTML5 and CSS3
